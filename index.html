Thread Navigator is a project by Dan Goodman to experiment with different ways to display deeply nested social media threads. At the moment, it is working with Bluesky and only has a linear, hierarchical thread view. I'll be adding Mastodon support and different views over time. Check out the project and help out at https://github.com/thesamovar/thread-navigator. You can create a bookmarklet in your browser with the following code as the URL so that when you're navigating on Bluesky and click the bookmarklet, you'll be taken to a thread view of the current page:
javascript:(() => {const url = encodeURIComponent(window.location.href); window.open(`https://thesamovar.github.io/thread-navigator/index.html?url=${url}`, '_blank');})()
